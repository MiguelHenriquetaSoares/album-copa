# Álbum da Copa

## Aluno

Miguel Henriqueta Soares

## Curso

Técnico em Informática para Internet

## Descrição

Aplicativo desenvolvido em Ionic Vue para gerenciamento de um álbum de figurinhas da Copa do Mundo.

## Funcionalidades

* Login
* Cadastro
* Recuperação de senha
* Visualização de figurinhas
* Controle de figurinhas coletadas
* Coleção de figurinhas

## Tecnologias

* Ionic
* Vue 3
* TypeScript

## Como executar

```bash
npm install
ionic serve
```
